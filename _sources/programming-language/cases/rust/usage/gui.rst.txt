其他系列
========================================

* `OrbTk (Orbital Widget Toolkit) <https://github.com/redox-os/orbtk>`_
    - 由 Redox OS 社群開發
    - 基於 ECS (Entity Component System)
    - 支援許多平臺
* `Druid - data-first Rust-native UI toolkit <https://github.com/linebender/druid>`_
    - 支援許多平臺
    - 根據不同平臺選擇後端，例如 Linux 可以用 GTK，macOS 可以用 Cocoa
* `Tauri <https://tauri.studio/>`_
    - 根據不同平臺選擇不同 WebView，例如 Linux 是 gtk-webkit2，macOS 是 Cocoa/WebKit
